# my-gui-port-scanner
## Small GUI port scanner :
- Use Nmap to discover service on open port.
- Generate Bash an Python reverse shell with your ip
- Hash identifier

```bash
pip install -r requirements.txt
```
```bash
python3 gui-portscanner.py
```
